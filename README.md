# TimeTracker2

A modern, dark-themed Timesheet Tracker built with .NET 10.0 and Windows Forms.

## Overview
TimeTracker2 is a desktop application designed to help users track their daily tasks and projects with a clean, modern user interface. It features a custom card-based design, automated login, and persistent data storage.

## Key Features
- **Modern UI/UX:** Dark-themed interface using a navy and slate color palette with sky blue accents.
- **Custom Controls:** Rounded panels and custom-drawn ListBoxes for a modern look and feel.
- **Smart Login:** Validation-aware login system that remembers your session and auto-redirects to the Main Menu.
- **Persistent Storage:** Automated file-based data management in the user's Documents folder.
- **Task Management:** A streamlined view for managing and selecting ongoing tasks.

## Getting Started
1. Clone the repository.
2. Open the solution in Visual Studio or Cursor.
3. Build and run the project.
4. Enter your name on the first launch to initialize your profile.
